## Описание
Компонент для локального управления устройствами [ZONT](https://zont-online.ru/) из Home Assistant. 

Интеграция поддерживает все устройства у которых есть локальный web интерфейс.

* Укажите любое удобное название устройства.
* При добавлении устройства в Home Assistant необходимо ввести полный адрес от web интерфейса
с эндпоитом `/ws`. 
```
https://192.168.1.40/ws
```
> [!WARNING]
> Обратите внимание на протокол http или https

* Логин и пароль из интерфейса ZONT.
```
По умолчанию 
login: admin
password: admin
```

## Возможности
В НА из ZONT добавляются следующие объекты каждого устройства:
* [Контуры отопления](#контуры-отопления)
* [Сенсоры](#сенсоры)
* [Элементы управления](#кастомные-элементы) (кнопка, сложная кнопка или бинарный сенсор)
* [Охранные зоны](#охранные-зоны)

### Контуры отопления.
Контур отопления добавляется в НА в виде термостата. Шаг регулировки температуры 0.5 градуса. 
Все, созданные в ZONT, названия режимов отопления добавляются в НА. 
Для изменения режима отопления во всех контурах одновременно в НА добавляются кнопки с названиями режимов.
> [!WARNING]
> Индивидуальная смена режима контура не реализована в локальном управлении.
> Только путём смены во всех контурах с помощью кнопки режима.
> Включение и выключение контура возможно только через режимы отопления. 
> Управление контуром в НА через set_hvac_mode не предусмотрено. 
> Такова особенность ZONT.

### Сенсоры.
В НА добавляются: 
* Проводные сенсоры температуры (цифровыве и аналоговые)
* Радио сенсоры (температура, влажность, движение, протечка)
* Аналоговые сенсоры
* Бинарные сенсоры
* Сенсоры цифровой шины котла (Температура теплоносителя, ГВС, модуляция, статус работы котла, код ошибки, давление)
* Сервисные сенсоры
  * Уровень сигнала WI-FI с атрибутами:
    * IMEI
    * Номер телефона
    * Оператор связи
  * Уровень сигнала GSM с атрибутами:
    * Маска 
    * Шлюз
    * MAC адрес
    * IP адрес
  * Статус подключение к локальной сети с атрибутами:
    * Маска 
    * Шлюз
    * MAC адрес
    * IP адрес
  * Статус подключения к облаку ZONT
  * Напряжение питания контроллера

Обновление измерительных сенсоров происходит примерно раз в минуту. 
Бинарные сенсоры обновляются по событию.

``*Для добавления необходимых сенсоров пишите в телеграм``

### Элементы управления.
В веб интерфейсе ZONT можно добавить элементы управления и привязать на них любое доступное действие.
Например, включить реле или выключить реле, включить котловой режим, включить отопительный режим на всех устройствах и др.
Смотрите возможности в интерфейсе ZONT.
Устройства, у которых тип элемента - статус входа/выхода, добавляются как бинарные сенсоры.


> [!WARNING]
> Все реле и насосы добавляются интеграцией только через элементы управления.
> Так все состояния будут меняться мгновенно.
 

В интерфейс добавляется кнопка перезагрузки контроллера.


### Охранные зоны.

В НА добавляются все охранные зоны устройства. Можно поставить на охрану или 
снять с охраны охранную зону.


## Установка
**Способ 1.** [HACS](https://hacs.xyz/) -> Интеграции -> 3 точки в правом верхнем углу -> Пользовательские репозитории

Далее вставляем репозиторий https://github.com/MihVS/zont_ws выбираем категорию "Интеграция" и жмём добавить.
***
**Способ 2.** Вручную скопируйте каталог `zont_ws` в директорию `/config/custom_components`
***
### **Не забываем поставить ★ интеграции. Это важно для меня.**

## Настройка
> Настройки -> Интеграции -> Добавить интеграцию -> **ZONT**
 
Если интеграции не появилось в списке, то очистите кэш в браузере.

### Логирование.
Чтобы изменить уровень логирования, для выявления проблем, необходимо в файле `configuration.yaml` добавить:
```yaml
logger:
  logs:
    custom_components.zont_ws: debug
```
